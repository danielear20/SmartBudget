# SmartBudget

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

## Overview
SmartBudget is a modern and intuitive personal finance application designed to help you track your expenses, set monthly budgets, and gain insights into your spending habits with interactive charts and reports.

Visit the project repository here: `https://github.com/danielear20/SmartBudget`

## Features
- Add, edit, and delete expenses effortlessly
- Create and manage multiple budgets
- Visualize spending with dynamic charts
- Monthly reports and summaries
- Responsive design for desktop and mobile

## Tech Stack
- React.js for the frontend UI
- Node.js with Express for the backend API
- MongoDB for data storage
- Chart.js for data visualization

## Installation
1. Clone the repository:
```
git clone https://github.com/danielear20/SmartBudget.git
```
2. Navigate to the project directory:
```
cd SmartBudget
```
3. Install dependencies for server and client:
```
cd server && npm install
cd ../client && npm install
```
4. Set up environment variables for server (e.g., MongoDB URI).
5. Run backend server:
```
cd ../server && npm start
```
6. Run frontend client:
```
cd ../client && npm start
```

## Usage
- Open your browser and visit `http://localhost:3000`
- Register a new account or login with existing credentials
- Start adding your expenses and budgets
- Navigate through various reports and dashboards to monitor finances
